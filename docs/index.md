# python-exoscale library documentation

This provides Python bindings for the [Exoscale](https://www.exoscale.com/) cloud platform API.

```{note}
This library maintains compatibility with actively supported Python
versions: Python 3.9+. Older Python versions may be compatible but such
compatibility is not guaranteed.
```

## Installation

Install the package using pip:

```bash
pip install exoscale
```
